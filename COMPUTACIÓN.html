COMPUTACIÓN

El curso de computación es la carrera técnica que prepara a las personas para utilizar la computación para el desarrollo de sus diversas actividades que pueden ser educativas, tecnológicas o científicas. El curso está conformado por los siguientes niveles y cursos:

  • CICLO:I
    1. Microsoft Windows
    2. Microsoft Word
    3. Microsoft PowerPoint
  • CICLO:II
    1. Microsoft Access
    2. Páginas Web
    3. Internet
  • CICLO:III
    1. Microsoft Excel I
    2. Microsoft Excel II
    3. Paquetes Contables
    4. Aplicaciones Gerenciales
